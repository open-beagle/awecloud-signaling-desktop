# AWECloud Desktop 用户手册

## 简介

AWECloud Desktop 是一个桌面客户端应用，用于安全地访问通过 AWECloud Signaling 系统暴露的远程服务。

## 系统要求

- **操作系统**: Windows 10/11（64位）
- **内存**: 至少 512MB 可用内存
- **网络**: 能够访问 AWECloud Server

## 安装

### Windows

1. 下载 `awecloud-desktop-setup.exe`
2. 双击运行安装程序
3. 按照向导完成安装
4. 从开始菜单或桌面快捷方式启动应用

## 快速开始

### 1. 登录

首次启动应用时，您会看到登录界面：

1. **服务器地址**: 输入 AWECloud Server 的地址（例如：`server.example.com:8080`）
2. **Client ID**: 输入您的用户名或邮箱
3. **Client Secret**: 输入您的密钥
4. 点击"登录"按钮

> **提示**: 如果您没有账号，请联系管理员创建。

### 2. 查看服务列表

登录成功后，您会看到可访问的服务列表。每个服务卡片显示：

- **服务名称**: 服务的标识名称
- **Agent**: 提供服务的 Agent 名称
- **远程端口**: 服务在远程主机上的端口
- **描述**: 服务的简要说明
- **状态**: 当前连接状态（未连接/连接中/已连接/错误）

### 3. 连接服务

要连接到一个服务：

1. 在服务卡片中输入**本地端口**（例如：3306）
2. 点击"连接"按钮
3. 等待连接建立（状态变为"已连接"）
4. 现在您可以通过 `localhost:本地端口` 访问远程服务

**示例**：连接到远程 MySQL 数据库

```bash
# 假设您将本地端口设置为 3306
mysql -h 127.0.0.1 -P 3306 -u username -p
```

### 4. 断开连接

要断开服务连接：

1. 找到已连接的服务卡片
2. 点击"断开连接"按钮
3. 连接将被关闭，本地端口将被释放

### 5. 刷新服务列表

如果管理员添加了新服务或修改了权限：

1. 点击右上角的"刷新"按钮
2. 服务列表将更新

### 6. 退出登录

要退出当前账号：

1. 点击右上角的"退出登录"按钮
2. 所有连接将被断开
3. 返回登录界面

## 常见使用场景

### 场景 1: 访问远程数据库

1. 登录 Desktop 应用
2. 找到数据库服务（例如：MySQL、PostgreSQL）
3. 连接服务，使用本地端口 3306
4. 使用数据库客户端连接到 `localhost:3306`

### 场景 2: 访问远程 Web 服务

1. 登录 Desktop 应用
2. 找到 Web 服务
3. 连接服务，使用本地端口 8080
4. 在浏览器中访问 `http://localhost:8080`

### 场景 3: 访问远程 SSH

1. 登录 Desktop 应用
2. 找到 SSH 服务
3. 连接服务，使用本地端口 2222
4. 使用 SSH 客户端连接：`ssh user@localhost -p 2222`

## 配置文件

Desktop 应用的配置文件存储在：

- **Windows**: `%APPDATA%\awecloud-desktop\config.json`

配置文件包含：
- 上次使用的服务器地址
- Client ID（不包含密钥）

## 故障排查

### 问题 1: 无法登录

**可能原因**:
- 服务器地址错误
- Client ID 或 Secret 错误
- 网络连接问题
- Server 未运行

**解决方法**:
1. 检查服务器地址是否正确
2. 确认 Client ID 和 Secret
3. 检查网络连接
4. 联系管理员确认 Server 状态

### 问题 2: 连接服务失败

**可能原因**:
- 本地端口已被占用
- Agent 离线
- 网络问题

**解决方法**:
1. 尝试使用不同的本地端口
2. 联系管理员检查 Agent 状态
3. 检查网络连接

### 问题 3: 连接突然断开

**可能原因**:
- 网络中断
- Server 或 Agent 重启
- 超时

**解决方法**:
1. 检查网络连接
2. 重新连接服务
3. 如果问题持续，联系管理员

### 问题 4: 本地端口被占用

**错误信息**: "端口已被占用"

**解决方法**:
1. 使用不同的本地端口
2. 或关闭占用该端口的程序

## 安全建议

1. **保护您的 Client Secret**
   - 不要与他人分享
   - 定期更换（联系管理员）

2. **使用安全的网络**
   - 避免在公共 WiFi 上使用
   - 或使用 VPN

3. **及时断开连接**
   - 不使用时断开服务连接
   - 退出应用时会自动断开所有连接

4. **保持应用更新**
   - 定期检查更新
   - 安装最新版本
